jsSHA is a JavaScript implementation of the entire family of SHA hashes as defined in FIPS PUB 180-4 and FIPS PUB 202 (SHA-1, SHA-224, SHA3-224, SHA-256, SHA3-256, SHA-384, SHA3-384, SHA-512, SHA3-512, SHAKE128, and SHAKE256) as well as HMAC as defined in FIPS PUB 198-1.
jsSHA is cross-browser compatible and works with Node.js.

Usage

Include the sha.js JavaScript file in the header of your page:
<script src='/path/to/sha.js'></script>

- Use the following syntax to get the desired hash type: 
// SHA-Type: SHA-1, SHA-224, SHA3-224, SHA-256, SHA3-256, SHA-384, SHA3-384, SHA-512, SHA3-512, SHAKE128, SHAKE256
// Input_Type: HEX, TEXT, BYTES, ARRAYBUFFER, B64 (Base-64)

var sha_ob = new jsSHA('SHA-Type', 'Input_Type');
sha_ob.update('Add your text');
var hash = sha_ob.getHash('HEX');
1. Instantiate a new jsSHA object (here sha_ob), using new jsSHA() with the desired hash type ('SHA-Type'), and input type ('Input_Type').
- The hash type ('SHA-Type') can be one of: SHA-1, SHA-224, SHA3-224, SHA-256, SHA3-256, SHA-384, SHA3-384, SHA-512, SHA3-512, SHAKE128, SHAKE256.
- The input type can be one of: HEX, TEXT, BYTES, ARRAYBUFFER, B64 (B64 is for Base-64).
2. Then stream in input using the update object function, sha_ob.update() with the text you want {you can call it multiple times if it is needed to append other text).
3. Finally, call sha_ob.getHash() with the output type as its argument (B64, HEX, BYTES, ARRAYBUFFER).

HMAC

var sha_ob = new jsSHA('SHA-Type', 'Input_Type');
sha_ob.setHMACKey('my_key', 'TEXT');
sha_ob.update('Add your text');
var hmac = sha_ob.getHMAC('HEX');
1. Instantiate a new jsSHA() object the same way as for hashing.
2. Then set the HMAC key to be used by calling sha_ob.setHMACKey() with the key (here 'my_key'), and its input type (here TEXT). This MUST be done before calling sha_ob.update().
3. Then stream in the input using the update object function just like hashing, sha_ob.update() with the text you want.
4. Finally, get the HMAC by calling the sha_ob.getHMAC() method with the output type as its argument (B64, HEX, BYTES, ARRAYBUFFER).

setHMACKey() takes the same input types as the constructor (jsSHA() ).
getHMAC() takes the same inputs as getHash().

- You cannot calculate both the hash and HMAC using the same object.

Example to calculate the SHA-512 HMAC of the string "Peace is Good" with the key "ab3": 
var sha_ob = new jsSHA('SHA-512', 'TEXT');
sha_ob.setHMACKey('ab3', 'TEXT');
sha_ob.update('Peace is Good');
var hmac = sha_ob.getHMAC('HEX');

alert(hmac)
